# Minerva

Minerva is a privacy-first mobile medical platform built using Flutter and Firebase that provides a secure, anonymous gateway for women facing sociocultural taboos, family constraints, or modesty barriers to consult verified female doctors safely.

## Structure

This is a monorepo containing:
- `apps/mobile`: The Flutter mobile application.
- `apps/web`: The React admin dashboard.
- `backend/functions`: Firebase Cloud Functions.
- `packages/shared`: Shared libraries and types.
- `docs`: Project documentation.
